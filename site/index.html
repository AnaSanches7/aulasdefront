Desenvolvimento de Sistemas

Formação técnica para o futuro da tecnologia

Sobre o Curso

O curso técnico em Desenvolvimento de Sistemas prepara o aluno para atuar com programação, análise de sistemas, banco de dados e desenvolvimento web. Com foco prático e atualizado, é ideal para quem deseja ingressar rapidamente no mercado de trabalho ou continuar os estudos na área de TI.

Conteúdo Programático

  • Fundamentos da Computação
  • Lógica de Programação e Algoritmos
  • Desenvolvimento Web (HTML, CSS, JavaScript)
  • Banco de Dados e SQL
  • Programação Orientada a Objetos (Java e/ou Python)
  • Modelagem de Sistemas e UML
  • Segurança da Informação
  • Projeto Integrador
