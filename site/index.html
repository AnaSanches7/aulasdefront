Formação técnica para o futuro da tecnologia

Sobre o Curso

O curso qualifica os estudantes em desenvolver o pensamento lógico e, tem como objetivo, de ampliar a area funcional da informatica na preparação para o mercado de trabalhar.

Conteúdo Programático

  • Analise e Projeto de Sistemas
  • Banco de Dados
  • Ciência da Computação
  • Lógica Computacional
  • Programação Back-end
  • Programação Front-end
  • Programação Mobile
  • Programação no Desenvolvimento de Sistemas
  • Ciência de Dados
  • Computação Gráfica

Benefícios do Curso

Certificação Técnica

Ao finalizar o curso, nesses três anos cursando integralmente o ensino médio, a coordenação se responsabiliza em certificar cada aluno que finalizou, completamente, o curso em todo o seu período de estudos.

Professores Especialistas

Os professores são formados em suas respectivas áreas do curso e garatem preparar os alunos para o mercado de trabalho.
